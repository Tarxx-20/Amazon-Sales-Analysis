# Amazon-Sales-Analysis
**📊 Amazon Sales Analysis Dashboard Overview**

This project presents an interactive Amazon Sales Analysis Dashboard built in Microsoft Excel to analyze sales performance, customer purchasing trends, product categories, payment methods, and order fulfillment metrics.

The dashboard transforms raw transactional data into actionable business insights using Pivot Tables, Pivot Charts, KPIs, and data visualization techniques.

**Dashboard Features** **:**
Interactive KPI Cards

Monthly Sales Analysis

Sales by Product Category

Country-wise Order Distribution

Payment Method Analysis

Delivery Status Monitoring

Dynamic Pivot Tables & Charts

**Tools & Skills Used :**

Microsoft Excel

Pivot Tables & Pivot Charts

Dashboard Design

Data Cleaning & Transformation

KPI Development

Business Intelligence & Data Analysis

**Project Outcome :**

The dashboard enables quick identification of sales trends, customer behavior patterns, top-performing categories, and operational performance, supporting data-driven business decisions.
